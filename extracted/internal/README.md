# Third-party dependencies (not redistributed)

This directory is populated by `setup_dependencies.sh`, which fetches, from the
labiia_lex repository (Rafael Cardoso Sampaio, GNU GPL v3), the IRaMuTeQ dictionaries
and the canonical R scripts required by the pipeline:

- `dictionaries/` (lexique_pt.txt, key.cfg, expression_pt.txt, ...)
- `Rscripts/` (CHD.R, chdfunct.R, anacor.R, simi.R, ...)

These components are third-party and are not part of this repository's original code.
See the top-level `NOTICE` for attribution.
